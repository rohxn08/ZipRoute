# Delivery Route Optimizer

A comprehensive solution for optimizing delivery routes using AI, providing real-time tracking, and efficient logistics management.

- **[backend/](backend/README.md)**: Python FastAPI server handling logic, database, and AI processing.
- **[frontend/](frontend/README.md)**: Flutter application for drivers and admins.
